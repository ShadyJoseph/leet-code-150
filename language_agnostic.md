## How to make ANY solution language-agnostic

Use this mental template:

### Step 1: Describe the data structure

array = contiguous memory block
linked list = node chain
hash map = key-value lookup

### Step 2: Describe allowed operations

access
update
move/shift
insert/delete

### Step 3: Describe transformations in words NOT code.

Example:

“Take last k elements and move them to the front.”
